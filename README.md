# dictionary
A dictionary in python and bash❣️. 
# Installation
```source <(curl -fsSL "https://git.io/JuXRy") -i```

# usage
- To use it just run:-
```dictionary```

- To update it just run:-
```dictionary -u``` or ```dictionary --update```

- For hard update just run:-
```dictionary ~u``` or ```dictionary ~~update```
